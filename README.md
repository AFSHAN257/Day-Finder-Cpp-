# Day Finder in C++

A simple C++ program that finds the day of the week for any given date.

## Features
- Correct leap year handling (4 / 100 / 400 rule)
- Accurate February adjustment
- Uses modular arithmetic to calculate weekdays
- Clean console output

## How to Run
```bash
g++ day_finder.cpp
./a.out
